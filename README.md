# Dice Roller (JavaScript)

A simple JavaScript dice roller that allows users to roll multiple dice and see the results both as numbers and images.

## Overview
This project was created to practice JavaScript concepts such as DOM manipulation, event handling, and dynamic content rendering.

## Features
- Roll one or multiple dice
- Generate random values between 1 and 6
- Display results as text and dice images
- Input validation with a reasonable limit on the number of dice

## Technologies
- JavaScript
- HTML
- CSS

## Usage
1. Open the `index.html` file in your browser.
2. Enter the number of dice you want to roll.
3. Click the **Roll Dice** button.
4. View the results displayed as numbers and images.
